# Export-WinRM

## SYNOPSIS

Export WinRM configuration to file

## SYNTAX

```powershell
Export-WinRM [<CommonParameters>]
```

## DESCRIPTION

Export-WinRM exports current WinRM configuration to file to be able to import
settings later and restore changes

## EXAMPLES

### EXAMPLE 1

```powershell
Export-WinRM
```
