# Get-SQLInstance

## SYNOPSIS

Retrieves SQL server information from a local or remote servers.

## SYNTAX

```none
Get-SQLInstance [[-ComputerName] <String[]>] [-CIM] [<CommonParameters>]
```

## DESCRIPTION

Retrieves SQL server information from a local or remote servers.
Pulls all
instances from a SQL server and detects if in a cluster or not.
