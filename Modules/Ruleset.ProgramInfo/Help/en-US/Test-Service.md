# Test-Service

## SYNOPSIS

Check if service exists on system

## SYNTAX

```powershell
Test-Service [-Service] <String> [<CommonParameters>]
```

## DESCRIPTION

Check if service exists on system, if not show warning message

## EXAMPLES

### EXAMPLE 1

```powershell
Test-Service dnscache
```

## PARAMETERS
