
TOPIC
	about_Ruleset.Test

SHORT DESCRIPTION
	Functions to write powershell unit tests

LONG DESCRIPTION
	Specialized functions designed for unit testing of "Windows Firewall Ruleset" project

EXAMPLES
	Enter-Test
	Initialize unit test

	Start-Test
	Start test case

	Stop-Test
	Stop test case

	Exit-Test
	Un-initialize and exit unit test

	New-Section
	Print new unit test section

	Test-Output
	Verify TypeName and OutputType are referring to same type
